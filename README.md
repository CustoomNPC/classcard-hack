## 📄 사용법
1. index.ts 컴파일.
2. `node index.js` 후 생성된 config.json 파일 수정.
3. `node index.js`
4. 봇을 디스코드 서버에 초대.
5. 디스코드 서버에서 `!setup`을 전송.
6. `#사용`채널에서 봇이 하라는 데로 하면 됨.
